## How to use this index

1. **Need to consult a manual?** Look up the component below, check the "Local file" column. If filled, open the PDF in [`manuals/`](manuals/). If blank, follow the Source URL and download it.
2. **Adding new gear?** Add a row to the appropriate section with vendor URL.
3. **PDF naming convention** is documented in [`manuals/README.md`](manuals/README.md).
4. **Source URLs may rot** — if a link 404s, search the manufacturer support page for the model number.

## Notes

- URLs are best-known stable manufacturer support pages, not necessarily direct PDF links — direct PDF URLs change frequently.
- For discontinued products (Marantz SR5014, KEF Q150, KEF Q500, LG 75UN7370PUE), keep the PDF locally even if the support page eventually goes dark.
- Banana plugs / Amazon-Basics cables / silicone pads typically don't have manuals; entries are blank intentionally.
